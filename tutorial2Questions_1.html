Question 1

Inline styles are style rules written as attributes inside an element. Due to this, the style rule only applies to a single element.

Embedded styles are style rules written inside the head of an HTML document using the "style" tag. These rules will only apply to the document it is written in.

External style sheets are style rules written inside a CSS file and then linked to the HTML document. External style sheets can be applied to many different web pages which allows the programmer to make changes to many web pages by editing a single CSS file.

Question 6

Progressive enhancement is the process of writing code from older versions of a programming language before code from more recent versions. Consequently, the older code is used by the web browser first and the newer code is used last. This allows older browsers to still run the files properly but also allows browsers that support more recent versions of the language to implement the newer standards and techniques.

Question 7

The older browser would use the color displayed by the rgb(232, 121, 50) color value because the rgb color value is supported by older versions of CSS whereas the hsl color value comes from more recent versions of CSS and would not be supported by older browsers. Furthermore, the older rgb value is written before the hsl value and therefore takes advantage of progressive enhancement.
